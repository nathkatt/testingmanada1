Significado de las características de los paquetes:

01
Secciones

Cantidad de páginas de información que tendrá tu sitio. Por ejemplo: Inicio, Nosotros, Servicios, Contacto, etc.

02
Cambios

Número de cambios que el cliente puede realizar a la página final.

03
Chat, formulario, maps

Contamos con métodos de contacto para que el usuario pueda contactarse rápidamente.

04
Certificado SSL

Certificamos el sitio web con SSL para que tu página sea segura para tus usuarios.

05
Correos corporativos

Configuramos correos empresariales para que tu marca sea aún más profesional.

05
Sitio Responsivo

Usamos código optimizado para hacer que el sitio sea lo más rápido posible y se visualice en cualquier dispositivo.

06
Hosting

Hospedamos tu sitio en la plataforma más segura que existe.

07
Dominio

Configuramos tu dominio, el cual es el URL donde encontrarás tu sitio web.

08
Mantenimiento y Administración

Soporte, ajustes, cambios (dentro de tiempos establecidos).

09
Actualizaciones técnicas

Modificaciones del sistema y del servidor.
